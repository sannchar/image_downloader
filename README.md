# Image Paster - Liquid Glass Desktop App

Stylish desktop application to download images from URL or clipboard with format conversion.

## Features
- **Liquid Glass Design**: Modern aesthetics with glassmorphism and animated backgrounds.
- **URL Download**: Paste direct image link to preview and download.
- **Clipboard Support**: Press `Ctrl+V` to paste images directly from your clipboard.
- **Format Conversion**: Convert images to PNG, JPG, or WEBP on the fly.
- **Native OS Dialog**: Robust saving using Windows File System Access API.

## How to Run
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Clone the repository.
3. Run `npm install` to install dependencies.
4. Run `npm start` (or use `start.bat`) to launch the application.
